# DomainIntel

DomainIntel is a Python-based domain intelligence and reconnaissance tool that automates information gathering about public domains.

## Features

* Domain to IP Resolution
* WHOIS Lookup
* DNS Enumeration
* SSL Certificate Inspection
* Security Header Analysis
* Automated Report Generation

## Technologies

* Python
* Requests
* dnspython
* python-whois
* SSL
* Socket

## Example Output

* Domain Information
* Registrar Details
* DNS Records
* SSL Certificate Information
* Security Header Assessment
* Security Grade
